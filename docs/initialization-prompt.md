# 苏州启樾投资有限公司项目初始化 Prompt

请为“苏州启樾投资有限公司”创建一个长期使用的公司经营操作系统项目。该项目不是普通资料夹，而是一个面向早期创业公司经营管理、项目孵化、商业计划管理和关系维护的轻量化 Business Operating System。

## 项目目标

1. 统一管理公司的创业项目、商业计划书、初始想法、成熟商业方案、运营事项、员工关系和社会关系。
2. 支持从“零散想法”到“项目立项”再到“成熟商业方案/BP”的递进管理。
3. 支持创始人日常经营复盘、会议纪要、决策追踪和下一步行动管理。
4. 支持关键员工、合作伙伴、社会关系、政府/机构关系等关系资产的长期维护。
5. 同时提供 Markdown 经营知识库、结构化数据文件和本地可交互前端看板。

## 交付要求

- 使用中文作为默认文档语言，表达专业、克制、可执行。
- 创建 `README.md`、`index.html`、`assets/`、`data/`、`templates/`、`scripts/` 和经营知识库目录。
- 前端页面必须从 `data/*.json` 读取内容，不把业务数据硬编码在 HTML 中。
- 页面第一屏展示公司名称、定位、经营重点和关键指标。
- 支持查看 Dashboard、创业项目、商业计划、想法池、运营节奏、关系维护、会议决策。
- 敏感信息不得写入真实数据，只使用占位说明。

## 验收标准

- `npm run serve` 可以启动本地前端看板。
- `npm test` 可以验证项目关键文件和数据结构。
- README 能说明如何新增项目、更新数据、记录会议和维护关系。
- 初始数据为示例占位，不包含真实隐私或未公开商业机密。
