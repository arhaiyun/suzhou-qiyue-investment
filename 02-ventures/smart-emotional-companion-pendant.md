# 实体智能毛绒挂件与云端数字生命体

## 1. 项目摘要

- 项目名称：实体智能毛绒挂件与云端数字生命体
- 当前阶段：机会识别 / Demo 方案验证
- 负责人：创始人
- 创建日期：2026-06-10
- 最近更新：2026-06-10
- 源材料：`docs/source-materials/项目工作流输入清单.docx`

本项目拟开发一款“实体陪伴型智能挂件 + 云端数字生命体”的软硬件结合产品。实体部分是可随身携带的毛绒互动挂件，例如尾巴、耳朵、毛球或原创异形生物部件；云端部分是完整 2D 数字生命体，承载角色人格、互动记忆、故事流、虚拟养成和后续商业化内容。

一句话表达：

> 它是一条会有反应、会发热、会记住你，并在手机里长成一个完整小生命的智能尾巴。

## 2. 问题与机会

- 要解决的问题：年轻用户、独居用户和情绪陪伴需求人群需要一种比普通饰品更有回应、比普通玩具更有关系感、比纯 AI 聊天更可触摸的陪伴产品。
- 目标客户或服务对象：年轻女性、学生、初入职场人群、潮玩与包挂用户、虚拟宠物用户、AI companion 用户、海外成人情绪陪伴用户。
- 为什么现在值得做：AI 陪伴、潮玩、个性包挂、虚拟养成和软硬件互动正在形成交叉机会，团队又具备设计、供应链和 IP 协同基础。
- 已观察到的证据：源材料已明确产品形态、核心功能、目标用户、定价区间、Demo 路径和团队资源。

## 3. 方案假设

- 核心方案：用实体毛绒挂件承接触摸、捏压、抚摸、发热和动作反馈，用云端软件承接完整角色、成长故事、关系记忆和虚拟消费。
- 与现有方案的差异：不主打“会聊天的玩具”，而主打“可触摸的数字生命入口”；实体是情绪入口，云端是人格和故事系统。
- 启樾的资源或能力优势：产品审美与形象设计能力、苏州工业园区供应链协调能力、商业模型设计能力、潜在明星 IP 联名资源。
- 最小可验证动作：完成一个半功能实体 Demo 和一个软件故事 Demo，验证用户是否能感受到“触摸实体后，云端角色产生关系反馈”。

## 4. 商业模式

- 收入来源：硬件销售、软件会员、大模型陪伴订阅、虚拟服装与房屋、故事包、限定角色、IP 联名款、海外版本溢价。
- 定价假设：小型入门款 299-499 元；大型高配款 899-999 元。
- 成本结构：外观设计、结构工程、柔性传感器、温控模块、动作反馈模块、电池与安全设计、毛绒制造、软件开发、云端模型调用、IP 合作与渠道推广。
- 关键资源：原创角色设计、硬件工程样机、毛绒与智能硬件供应链、软件端角色系统、用户测试渠道。
- 合作方：柔性传感器企业、温控传感器企业、封装测试企业、毛绒玩具制造商、IP 方、内容创作者、种子用户社群。

## 5. 风险与约束

- 最大不确定性：用户是否愿意为“实体互动 + 云端故事关系”持续付费，而不是只把它视为新奇毛绒饰品。
- 技术风险：柔性传感器在毛绒材料上的稳定性、动作反馈结构耐久性、发热与电池安全、重量与续航平衡。
- 产品风险：容易被误解为“普通毛绒玩具加震动”，或被 AI 玩具同质化叙事淹没。
- 商业风险：硬件复杂度抬高 BOM 和售后压力，软件故事流如果不够强会影响留存。
- 合规约束：发热、电池、儿童与成人市场边界、海外销售、用户隐私和 AI 陪伴内容边界都需要提前评估。
- 暂停条件：样机成本无法进入目标价格带；核心互动无法形成情绪闭环；种子用户访谈显示购买理由不足；发热或电池安全无法满足认证要求。
